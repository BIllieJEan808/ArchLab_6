Общее описание

Проект представляет собой онлайн-библиотеку с возможностью:

    Регистрации и авторизации пользователей

    Просмотра доступных книг

    Аренды книг (взятие/возврат)

    Покупки книг

Технологический стек:

    Фронтенд: HTML5, CSS3, JavaScript

    Бэкенд: PHP 8.2

    База данных: PostgreSQL

    Инфраструктура: Docker, Apache

Фронтенд-часть
Структура интерфейса

    Страница авторизации (index.html):

        Форма входа с email/паролем

        Переключение на режим регистрации

        Валидация полей на клиенте

    Главная страница (main.html):

        Три карточки действий:

            "Взять книгу" (переход на take-book.html)

            "Вернуть книгу" (переход на return-book.html)

            "Купить книгу" (переход на buy-book.html)

    Страницы операций:

        Таблицы с книгами

        Кнопки действий (взять/вернуть/купить)

        Ссылка возврата на главную

Логика работы

    Все переходы между страницами выполняются без перезагрузки (SPA-подход)

    Данные пользователя сохраняются в localStorage после авторизации

    Интерактивные элементы имеют визуальную обратную связь (hover, active состояния)

    Формы обрабатываются через AJAX-запросы к API

Бэкенд-архитектура
Основные компоненты

    API Gateway (api.php):

        Обработка всех входящих запросов

        Маршрутизация на конкретные обработчики

        Базовая валидация данных

    Модель данных (db.php):

        Подключение к PostgreSQL

        PDO-обёртка для безопасных запросов

        Управление соединением с БД

    Бизнес-логика:

        Регистрация/авторизация пользователей

        Управление книгами (CRUD операции)

        Обработка транзакций (аренда/покупка)

Инициализация БД

Файл db/init.sql содержит:

    DDL-скрипты создания таблиц

    Индексы для часто используемых полей

    Тестовые данные (опционально)

Серверная часть
Конфигурация Apache

    Виртуальный хост на порту 80

    Два основных location:

        / - статические файлы фронтенда

        /api - PHP-скрипты бэкенда

    Правила перезаписи URL для:

        Поддержки HTML5 History Mode

        "Чистых" URL API

Docker-образ

Основан на php:8.2-apache с:

    Установленными расширениями pdo_pgsql

    Включённым mod_rewrite

    Настроенными правами доступа

Сборка и запуск
Требования

    Docker 20.10+

    Docker Compose 1.29+

Инструкция

    Клонируйте репозиторий:
    bash

git clone [ваш-репозиторий]
cd online-library

Настройте переменные окружения (файл .env):
ini

DB_USER=library_user
DB_PASSWORD=secret
DB_NAME=online_library

Соберите и запустите контейнеры:
bash

    docker-compose up --build -d

    Инициализация БД происходит автоматически при первом запуске

    Доступ к приложению:

        Фронтенд: http://localhost:8000

        База данных: localhost:5432 (логин/пароль из .env)

Команды управления

    Остановка: docker-compose down

    Просмотр логов: docker-compose logs -f web

    Пересборка: docker-compose up --build

API Endpoints
Пользователи

    POST /api?action=register - регистрация

    POST /api?action=login - авторизация

Книги

    GET /api?action=get_books - список всех книг

    POST /api?action=borrow - взять книгу

    POST /api?action=return - вернуть книгу

    POST /api?action=purchase - купить книгу

Пример запроса
bash

curl -X POST http://localhost:8000/api?action=register \
-H "Content-Type: application/json" \
-d '{"email":"test@example.com","password":"qwerty123","first_name":"Иван","last_name":"Иванов"}'
